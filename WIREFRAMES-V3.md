# Wireframes v3.0 - Complete Redesign

## Overview

Simplified, calendar-first design with icon-only sidebar navigation.

**Key Changes from v2:**
- Icon-only sidebar (60px) instead of text labels
- Home screen with nudges and quick actions
- Analytics page for usage tracking
- Improved settings organization
- Unified list + detail pattern across all views
