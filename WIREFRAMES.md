# Wireframes v2.0

ASCII wireframes for the meeting recorder app. Design inspired by Amie: minimal, clean, lots of whitespace, native macOS feel.

**Changes from v1.0:**
- Simplified calendar view (removed mini calendar)
- Narrower menu bar dropdown (300px max)
- Settings uses toolbar with icons
- Added error states
- Waveform on dictation indicator
- Dockable dictation pill
- Action items now editable
